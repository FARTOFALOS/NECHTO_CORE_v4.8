# NECHTO CORE v4.8 Examples

These examples are runnable against the current runtime. Install the package in editable mode first:

```bash
pip install -e .
```

## Examples
- `01_basic_cli.py`: Runs the CLI via `subprocess` and prints the written metrics JSON.
- `02_measure_and_print_metrics.py`: Calls the Python `measure_text` helper and prints key fields.
